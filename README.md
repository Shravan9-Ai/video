# Student Registration Form

## Overview

This is a simple **Student Registration Form** created using **HTML**. The form collects basic student information such as roll number, name, father's name, date of birth, mobile number, email, password, gender, department, course, photo, and address.

## Features

* Roll Number input
* Student Name input
* Father's Name input
* Date of Birth selection
* Mobile Number input
* Email ID input
* Password field
* Gender selection using radio buttons
* Department selection
* Course input
* Student Photo upload option
* Address input

## Technologies Used

* HTML5
